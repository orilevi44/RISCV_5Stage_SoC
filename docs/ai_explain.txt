

Hello! I am working on a custom 32-bit RISC-V System-on-Chip (SoC) written in SystemVerilog. The project features a 5-stage pipelined CPU core with full hazard detection and data forwarding, connected via a system bus to internal memory and peripherals. 

The project is fully verified, and the current state includes a working UART echo test. Before we start working on specific tasks, I want to provide you with the project's folder structure and architecture so you understand the context and separation of concerns.

### **Memory Map:**
* `0x0000 - 0x0FFF`: ROM (Instructions)
* `0x1000 - 0x1FFF`: GPIO
* `0x2000 - 0x2FFF`: RAM (Data)
* `0x3000 - 0x300F`: UART (0x3000: RX/TX Data, 0x3004: Status Register)
